# CRUD APPLICATION W/ NODE FOR BEGINNER

This is a CRUD Application with Node, Express & MongoDB

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install 
```
or

```bash
yarn install 
```
